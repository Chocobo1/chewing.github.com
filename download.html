Note / 注意
  • 新酷音是延續著名的「酷音輸入法」的發展計畫
  • 為了避免名稱混淆，本發展計畫特稱為「新酷音」
Tarball 下載

本專案釋出的檔案可在此路徑下載： http://chewing.csie.net/download/.

下載 Subversion 版本

  • svn co https://svn.csie.net/chewing
  • 瀏覽 Subversion 檔案庫

下載套件 (Pre-compiled and ports entries)

  • 提供給 Microsoft Windows 平台的 安裝程式 (setup.exe)，適用於 Windows 98/ME/2000/XP
  • Mandriva (前身為 Mandrake) Linux 已正式收錄 libchewing 與 SCIM-chewing
  • Debian GNU/Linux 已正式收錄 libchewing 與 SCIM-chewing，也可取得最新的非正式套件： APT /etc/apt/sources.list
      deb http://apt.debian.org.tw unstable main
  • SuSE (Official)
  • Fedora Core 已正式收錄 libchewing 與 SCIM-chewing
  • Gentoo Linux 已收錄 libchewing 與 SCIM-chewing，更新部份可參考 Gentoo Taiwan (GOT) 維護的 ebuild
  • FreeBSD 已收錄於 ports，可參見 FreeBSD chinese :: libchewing / scim-chewing
  • NetBSD 已收錄於 pkgsrc，可參見 scim-chewing
  • OpenBSD 已收錄於 ports，可參見 libchewing 與 scim-chewing
  • Sun Microsystems 的 JDS (Java Desktop System) 與 Solaris 已內建新酷音輸入法 (IIIMF-chewing / SCIM-chewing)，請更新軟體版本
