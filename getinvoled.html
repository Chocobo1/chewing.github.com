Build from Source Tarball

以 Debian GNU/Linux 為例，需要以下套件：
  • build-essential : C/C++ 開發環境
  • autoconf
  • automake
  • libtool
  • cmake ( 可選 )
  • libscim-dev : SCIM-chewing 所需 ( 可選 )
  • iiimf-dev : IIIMF-lichewing 所需 ( 可選 )
